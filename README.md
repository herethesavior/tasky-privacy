# Tasky — Privacy Policy

This repository hosts the privacy policy for **Tasky**, a task management app for Android.

**Live page:** https://herethesavior.github.io/tasky-privacy

---

## About Tasky

Tasky is an Android task manager with voice input, recurring tasks, reminders, and smart scheduling. All data is stored locally on your device — no accounts, no cloud, no tracking.

## About This Repo

This is a public repository solely for hosting the Tasky privacy policy as required by the Google Play Store. The app source code is maintained separately.

## Contact

For questions or concerns about the privacy policy, open an issue in this repository or visit [github.com/herethesavior](https://github.com/herethesavior).
